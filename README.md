### 👨‍💻 Tentang Saya

Halo! Saya seorang siswa kelas XI jurusan Rekayasa Perangkat Lunak (RPL) di **SMKN 4 Bandung**. Saya memiliki minat besar dalam pengembangan aplikasi bergerak dan web, serta sangat menyukai perancangan antarmuka yang bersih dan profesional.

- 🔭 Saat ini sedang fokus mengembangkan **LabReserve** (Aplikasi Reservasi Ruangan) dengan pemodelan sistem UML yang komprehensif.
- 🌱 Sedang memperdalam pengembangan aplikasi mobile menggunakan **Android Studio** dan web menggunakan framework **Laravel 11**.
- 💅 Selalu berusaha menerapkan prinsip UI/UX yang minimalis, terutama dengan skema warna *deep navy* dan *earth tones*.
- ⛰️ Di luar *coding*, saya menjaga rutinitas ke *gym* 3-4 kali seminggu, senang memasak, dan sesekali mendaki (Puncak Gunung Burangrang adalah salah satu favorit saya!).
- 💬 Tanya saya tentang: **Java, PHP, Perancangan Sistem, atau sastra/bahasa Sunda**.
- ⚡ Fakta unik: Saya menulis kode saya di MSI Thin GF63 12UCX, seringkali sambil mencari referensi *carpon* atau *sisindiran* untuk tugas!
